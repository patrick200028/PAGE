# Corretora Hozeni — Landing Page

Página de apresentação leve e responsiva para `Corretora Hozeni`, com formulário de captura de leads.

## Visão geral

Projeto simples: uma landing page estática em HTML/CSS, pensada para apresentar imóveis e captar contatos.

## O que foi feito
- Estrutura HTML semântica e acessível (`header`, `main`, `section`, `footer`).
- Estilo moderno e responsivo em `style.css` usando variáveis CSS e grid.
- Formulário de captura com feedback (alert) e foco em acessibilidade.

## Como usar localmente

1. Abra o arquivo `Untitled-1.html` no seu navegador.
2. Para desenvolvimento, edite `style.css` e `Untitled-1.html` conforme necessário.

## Publicar no GitHub

1. Crie um repositório no GitHub e faça commit dos arquivos.
2. Ative o GitHub Pages (Settings → Pages) apontando para a branch `main` (ou `gh-pages`).
3. O site ficará disponível em `https://<seu-usuario>.github.io/<seu-repo>/`.

## Personalização rápida
- Troque a imagem hero por imagens próprias.
- Integre um backend de formulário (ex: Formspree, Netlify Forms) no lugar do `alert`.
- Ajuste variáveis em `:root` do `style.css` para mudar cores e tipografia.

## Arquivos principais
- `Untitled-1.html` — HTML da página.
- `style.css` — Estilos principais.
